All cross-phase wiring between phases 8, 9, and 10 is properly connected. One integration gap found:

### Integration Issue: launch_game Missing +msgbox Default

**From:** Phase 10 `parseMsgboxDialogs()` → **To:** `launchGame()` in AgentTools.swift
**Affected:** DIAG-01, DIAG-03
**Severity:** Moderate

`launch_game` calls `parseMsgboxDialogs()` on stderr but does not enable `+msgbox` in WINEDEBUG by default. The `dialogs` array will always be empty from `launch_game` unless the agent explicitly sets the channel. `trace_launch` defaults to `["+loaddll", "+msgbox"]` and works correctly.

**Fix:** Add `+msgbox` to `launch_game` default WINEDEBUG — one-line change with negligible performance impact.
