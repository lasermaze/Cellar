<context>
The user's vision is an agent-first shared knowledge layer. When Agent A solves a game on one Mac, Agent B launching the same game elsewhere queries the collective memory, reasons about whether the stored config fits its local environment (hardware, Wine version, macOS), and adapts or applies accordingly. Confidence builds through multiple agents confirming configs work. The web interface should show the memory state.

The pre-installed game feature is a small quality-of-life fix — many old games come as extracted archives without installers.
</context>
