```
{thread_hex}:trace:msgbox:MSGBOX_OnInit L"{message_text_with_escapes}"
// Example: 0009:trace:msgbox:MSGBOX_OnInit L"Microsoft Mathematics has encountered a problem..."
// Wine only traces the message body text, NOT the caption or button type
```
