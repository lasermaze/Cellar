**Error enum:**
```swift
enum DiscImageError: LocalizedError {
    case hdiutilFailed(String)
    case plistParseFailed
    case noVolumesMounted
    case noInstallerFound
    case companionBinNotFound(String)
}
```
Each case should have a user-friendly `errorDescription` with "Try this:" suggestions per project convention.

**MountResult struct:**
```swift
struct MountResult {
    let mountPoint: URL
    let devEntry: String
    let tempConvertedISO: URL?  // non-nil when .bin was converted to temp ISO
}
```

**mount(imageURL:) throws -> MountResult:**
1. Check extension. If `.cue`, parse the CUE file to find the companion `.bin` filename (read `FILE "..." ...` directive), do case-insensitive search in same directory. If not found, throw `.companionBinNotFound`. Then treat as `.bin` path going forward.
2. For `.iso` (and `.img` — support as renamed .iso per discretion): run `hdiutil attach -readonly -nobrowse -plist <path>`. Parse plist output with PropertyListSerialization. Extract first entity with `mount-point` key from `system-entities` array. If no mount-point found, detach the dev-entry and throw `.noVolumesMounted` with message about unsupported format.
3. For `.bin`: First attempt `hdiutil attach -readonly -nobrowse -plist -imagekey diskimage-class=CRawDiskImage <path>`. If that fails (exit code != 0), attempt conversion: create temp dir via `FileManager.default.temporaryDirectory.appendingPathComponent("cellar-disc-\(UUID().uuidString)")`, run `hdiutil convert <bin-path> -format UDTO -o <temp-dir>/converted`. The output file will be `converted.cdr` — attach that instead. Store the temp dir URL in MountResult.tempConvertedISO for cleanup.
4. Return MountResult with mountPoint URL, devEntry string, and optional temp ISO path.

**Private helper runHdiutil(_ args:) throws -> Data:**
Run `/usr/bin/hdiutil` with given arguments via Foundation.Process. Capture stdout via Pipe. Capture stderr via separate Pipe. Wait for exit. If terminationStatus != 0, read stderr and throw `.hdiutilFailed(stderrString)`. Return stdout Data.

**Private helper parseMountInfo(from plistData:) throws -> (mountPoint: URL, devEntry: String):**
Parse plist with PropertyListSerialization. Walk `system-entities` array looking for first dict with both `mount-point` and `dev-entry` keys. If none found, check if any entity has a `dev-entry` (block device attached but no filesystem) — if so, throw `.noVolumesMounted` with actionable message. If no entities at all, throw `.plistParseFailed`.

**discoverInstaller(at mountPoint:) throws -> URL:**
Priority order per CONTEXT.md decisions:
1. Parse autorun.inf: check for "AUTORUN.INF", "autorun.inf", "Autorun.inf" at mountPoint root. Read with `.isoLatin1` encoding. Find line starting with `open=` (case-insensitive). Strip quotes, replace backslashes with forward slashes. Verify the file exists at mountPoint + path. Return if found.
2. Common names scan: iterate `["setup.exe", "install.exe", "Setup.exe", "Install.exe"]`. Return first that exists at mountPoint root.
3. All .exe fallback: use FileManager.contentsOfDirectory(at:mountPoint) to list all `.exe` files. If exactly one, return it. If multiple, print numbered list and prompt user to choose via `readLine()`. If zero, throw `.noInstallerFound`.

**detach(mountResult:):**
1. Run `hdiutil detach <devEntry>` via Process. Suppress stdout/stderr.
2. If terminationStatus != 0, retry with `hdiutil detach -force <devEntry>`.
3. If tempConvertedISO is non-nil, remove the temp directory with `try? FileManager.default.removeItem(at:)`.
4. This method must never throw — all errors caught with `try?`.

**volumeLabel(from mountPoint:) -> String?:**
Extract volume name from mountPoint.lastPathComponent. Return nil if it matches generic patterns: "CDROM", "CD_ROM", "DISC", "DISC1", "DISC_1", "DVD", "VOLUME", or is empty.
