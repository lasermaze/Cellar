Rewrite CollectiveMemoryWriteService to replace the direct GitHub API flow with a single POST to the Cloudflare Worker proxy. The file should shrink from ~384 lines to ~120 lines.

**Add proxy URL constant at the top of the struct:**
```swift
/// Proxy URL for collective memory writes.
/// Reads CELLAR_MEMORY_PROXY_URL from environment, falls back to production Worker.
private static var proxyURL: String {
    ProcessInfo.processInfo.environment["CELLAR_MEMORY_PROXY_URL"]
        ?? "https://cellar-memory-proxy.<your-account>.workers.dev/api/contribute"
}
```
Use a placeholder domain like `cellar-memory-proxy.cellar-community.workers.dev` — the user will update this after deploying the Worker.

**Rewrite push(record:gameName:wineURL:):**
1. Remove the auth check (`GitHubAuthService.shared.getToken()` call). Instead, just proceed directly.
2. Keep steps 2-6 (detect Wine version/flavor, build fingerprint, build entry) — these are unchanged.
3. Replace step 7 (the `pushEntry(entry:token:)` call) with a new `postToProxy(entry:)` call.

**New private static func postToProxy(entry: CollectiveMemoryEntry):**
- Build URL from `proxyURL`
- Create URLRequest with POST method
- Set Content-Type: application/json
- Encode `["entry": entry]` as JSON body (use a wrapper struct or JSONSerialization)
- Set timeoutInterval to 10 (proxy may be slower than direct API)
- Perform async request via URLSession
- Check response: 200/201 = success, 429 = rate limited (log and return), other = error (log and return)
- Log results using existing logPushEvent helper

**Rewrite syncAll(wineURL:):**
- Remove the `GitHubAuthService.shared.getToken()` check (lines 86-89). The proxy URL always has a default, so just proceed.
- Keep the rest of the logic (load records, iterate, push each).

**Delete entirely:**
- The `pushEntry(entry:token:url:)` private method
- The `performMergeAndPut(entry:token:url:)` private method
- The `MergeResult` enum
- The `GitHubContentsResponse` private struct
- The `performRequest(request:)` private method (replaced by inline URLSession call in postToProxy)

**Keep:**
- `push(record:gameName:wineURL:)` (rewritten)

- `detectWineVersion(wineURL:)` (unchanged)
