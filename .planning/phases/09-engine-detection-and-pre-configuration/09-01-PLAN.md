    - detectGraphicsApi() with peImports=["ddraw.dll","kernel32.dll"] returns "directdraw"

    - detectGraphicsApi() is case-insensitive: ["DDRAW.DLL"] returns "directdraw"

  <action>
Create Sources/cellar/Models/EngineRegistry.swift with:

1. `EngineDefinition` struct with fields: name (String), family (String), filePatterns ([String]), peImportSignals ([String]), stringSignatures ([String]), typicalGraphicsApi (String?).

2. `EngineDetectionResult` struct with fields: name (String), family (String), confidence (String — "high"/"medium"/"low"), signals ([String] — e.g. ["file:fsgame.ltx", "string:GSC Game World"]).

3. `EngineRegistry` with:
   - `static let engines: [EngineDefinition]` — all 8 families with patterns from RESEARCH.md code examples
   - `static func detect(gameFiles: [String], peImports: [String], binaryStrings: [String]) -> EngineDetectionResult?` — weighted scoring:
     - For each engine, check file patterns (case-insensitive). Unique files like "fsgame.ltx" or "UnityPlayer.dll" = +0.5. Common extensions like ".grp" or ".pak" = +0.3. Track matched signals as "file:filename".
     - Check PE import overlap with engine's peImportSignals = +0.25 per match. Track as "import:dll_name".
     - Check binary string matches (case-insensitive substring) = +0.15 per match. Track as "string:matched_text".
     - If multiple signal types agree, multiply total by 1.2.
     - Confidence thresholds: >= 0.6 = "high", >= 0.35 = "medium", >= 0.15 = "low". Below 0.15 = no match.
     - Return highest-scoring engine.
   - `static func detectGraphicsApi(peImports: [String]) -> String?` — case-insensitive DLL-to-API mapping: ddraw.dll="directdraw", d3d8.dll="direct3d8", d3d9.dll="direct3d9", d3d11.dll="direct3d11", opengl32.dll="opengl". Return first match in priority order (d3d11 > d3d9 > d3d8 > ddraw > opengl — prefer higher DX version if multiple present).

File pattern matching rules:
- Patterns starting with "*." match file extensions (case-insensitive suffix check)
- Patterns ending with "/" match subdirectory names
- Other patterns are exact filename matches (case-insensitive)
- Always lowercase both pattern and filename before comparison

Create Tests/cellarTests/EngineRegistryTests.swift testing all behaviors listed above.
