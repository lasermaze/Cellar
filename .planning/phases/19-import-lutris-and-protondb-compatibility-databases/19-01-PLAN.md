  <action>
Create `Sources/cellar/Core/CompatibilityService.swift` with:

**1. Lutris API Models (Codable):**
- `LutrisSearchResponse`: `{ count: Int, results: [LutrisGame] }`
- `LutrisGame`: `{ id: Int, name: String, slug: String, year: Int?, providerGames: [LutrisProviderGame] }` with CodingKeys for `provider_games`
- `LutrisProviderGame`: `{ name: String, slug: String, service: String }`
- `LutrisInstallerResponse`: `{ count: Int, results: [LutrisInstaller] }`
- `LutrisInstaller`: `{ id: Int, gameSlug: String, name: String, runner: String, script: LutrisScript? }` with CodingKeys for `game_slug`
- `LutrisScript`: `{ system: LutrisSystem?, wine: LutrisWineConfig?, installer: [LutrisTask]?, game: LutrisGameConfig? }`
- `LutrisSystem`: `{ env: [String: String]? }`
- `LutrisWineConfig`: `{ overrides: [String: String]? }`
- `LutrisTask`: `{ name: String?, app: String?, path: String?, key: String?, type: String?, value: String? }`
- `LutrisGameConfig`: Codable struct for the `game` key (optional fields, used as catch-all)

**2. ProtonDB Model:**
- `ProtonDBSummary`: `{ tier: String, bestReportedTier: String, trendingTier: String, confidence: String, score: Double, total: Int }`

**3. CompatibilityReport struct:**
- Fields: `gameName: String`, `lutrisSlug: String?`, `steamAppId: String?`
- Lutris extracted data: `lutrisEnvVars: [ExtractedEnvVar]`, `lutrisDlls: [ExtractedDLL]`, `lutrisWinetricks: [ExtractedVerb]`, `lutrisRegistry: [ExtractedRegistry]`, `installerCount: Int`
- ProtonDB data: `protonTier: String?`, `protonConfidence: String?`, `protonTotal: Int?`
- Computed `isEmpty: Bool` — true when all lists empty AND protonTier nil
- Method `func formatForAgent() -> String` that produces the formatted context block:
  ```
  --- COMPATIBILITY DATA ---
  Community compatibility data for '{gameName}':

  ## ProtonDB Rating
  Tier: {tier} (confidence: {confidence}, {total} reports)
  Trending: {trendingTier}

  ## Lutris Configuration (from {N} installer scripts)
  Environment variables:
    {var}={value}  (or "(none found)")
  DLL overrides:
    {dll} = {mode}  (or "(none found)")
  Winetricks:
    {verb1}, {verb2}  (or "(none found)")
  Registry edits:
    {keyPath}  {valueName} = {valueData}  (or "(none found)")

  Note: ProtonDB reports are from Linux+Proton users. Config hints above are filtered for Wine/macOS compatibility.
  --- END COMPATIBILITY DATA ---
  ```
  Omit the ProtonDB section entirely if protonTier is nil. Omit the Lutris section if installerCount == 0.

**4. Cache struct:**
- `CompatibilityCache<T: Codable>: Codable` with `fetchedAt: String` (ISO8601) and `data: T`
- `func isStale(ttlDays: Int = 30) -> Bool` — true if fetchedAt is more than ttlDays ago or unparseable

**5. CompatibilityService struct (static methods):**

- `static func fetchReport(for gameName: String) -> CompatibilityReport?`
  Main entry point. Steps:
  1. Search Lutris: `fetchLutrisGame(name:)` → best-match LutrisGame
  2. Extract Steam AppID from LutrisGame.providerGames (service == "steam", slug is the AppID)
  3. Fire parallel fetches: `fetchLutrisInstallers(slug:)` AND `fetchProtonDBSummary(appId:)` using DispatchSemaphore + ResultBox pattern (same as CollectiveMemoryService)
  4. Extract actionable config from Lutris installers: merge env vars, DLL overrides, winetricks verbs, registry edits across all Wine-runner installers
  5. Filter Proton-specific env vars (prefixes: PROTON_, STEAM_, SteamAppId, SteamGameId, LD_PRELOAD, WINEDLLPATH, WINELOADERNOEXEC, DXVK_FILTER_DEVICE_NAME)
  6. Build CompatibilityReport
  7. Return nil if report.isEmpty

- `private static func fetchLutrisGame(name: String) -> LutrisGame?`
  1. Check cache: `CellarPaths.lutrisCompatCacheDir/{normalizedName}.json`
  2. If cache miss or stale: GET `https://lutris.net/api/games?search={urlEncodedName}` with 5s timeout
  3. Score results with Jaccard token-overlap similarity against query name
  4. Require minimum similarity > 0.3 (use 0.3, not 0.5 — game names vary significantly between Cellar slugs and Lutris titles)
  5. Cache the best match under `{normalizedName}.json`
  6. Return best match or nil

- `private static func fetchLutrisInstallers(slug: String) -> [LutrisInstaller]`
  1. Check cache: `CellarPaths.lutrisCompatCacheDir/installers-{slug}.json`
  2. If cache miss or stale: GET `https://lutris.net/api/installers?game={slug}` with 5s timeout
  3. Cache result, return installers array (or empty on failure)

- `private static func fetchProtonDBSummary(appId: String) -> ProtonDBSummary?`
  1. Check cache: `CellarPaths.protondbCompatCacheDir/{appId}.json`
  2. If cache miss or stale: GET `https://www.protondb.com/api/v1/reports/summaries/{appId}.json` with 5s timeout
  3. Cache result, return summary or nil

- `private static func extractFromInstallers(_ installers: [LutrisInstaller]) -> (envVars: [ExtractedEnvVar], dlls: [ExtractedDLL], verbs: [ExtractedVerb], registry: [ExtractedRegistry])`
  Process only installers with runner == "wine". For each:
  - script.system?.env → map to ExtractedEnvVar (source: "lutris")
  - script.wine?.overrides → map to ExtractedDLL (source: "lutris")
  - script.installer tasks where name == "winetricks" → split app by spaces → map to ExtractedVerb
  - script.installer tasks where name == "set_regedit" → map to ExtractedRegistry
  Deduplicate by name/key across installers.

- Fuzzy matching helpers (private):
  - `normalizeGameName(_ name: String) -> Set<String>`: lowercase, strip punctuation, split on whitespace, filter empty, remove stop words ["the", "a", "an", "of", "and"]
  - `jaccardSimilarity(_ a: Set<String>, _ b: Set<String>) -> Double`: intersection.count / union.count (return 0 if both empty)

- Proton flag filtering (private):
  - `filterPortableEnvVars(_ vars: [ExtractedEnvVar]) -> [ExtractedEnvVar]`: filter out vars whose name starts with any prefix in the protonOnlyPrefixes list

- Cache helpers (private):
  - `readCache<T: Codable>(_ type: T.Type, from file: URL, ttlDays: Int) -> T?`: read file, decode CompatibilityCache<T>, return data if not stale
  - `writeCache<T: Codable>(_ data: T, to file: URL)`: create directory with intermediateDirectories, encode CompatibilityCache wrapping data with current ISO8601 timestamp, write to file

**Error handling:** All URLSession calls and JSON decoding wrapped in do/catch or optional chaining. Any failure returns nil/empty — NEVER throw from public methods. Log warnings to stderr with `print("[CompatibilityService]", to: &standardError)` pattern if available, otherwise just silently return nil.

**Sendable compliance:** Use the same `ResultBox<T>: @unchecked Sendable` pattern from CollectiveMemoryService for the parallel fetch boxes.
