**MemoryStatsService types (all must conform to Content for Leaf rendering):**

```swift
struct MemoryStats: Content {
    let gameCount: Int
    let totalConfirmations: Int
    let recentContributions: [RecentContribution]
    let isAvailable: Bool  // false when auth missing or network fails
}

struct RecentContribution: Content {
    let gameName: String
    let gameSlug: String
    let lastConfirmed: String
    let confirmations: Int
}

struct GameDetail: Content {
    let gameName: String
    let gameSlug: String
    let entries: [MemoryEntryViewData]
}

// Flat view model — Leaf cannot render computed properties or deeply nested optionals
struct MemoryEntryViewData: Content {
    let arch: String
    let wineVersion: String
    let macosVersion: String
    let wineFlavor: String
    let confirmations: Int
    let lastConfirmed: String
    let engine: String       // "" if nil
    let graphicsApi: String  // "" if nil
    let reasoning: String
}
```

**MemoryStatsService.fetchStats() -> MemoryStats:**
1. Get token via `GitHubAuthService.shared.getToken()`. If `.unavailable`, return `MemoryStats(gameCount: 0, totalConfirmations: 0, recentContributions: [], isAvailable: false)`.
2. GET `https://api.github.com/repos/\(GitHubAuthService.shared.memoryRepo)/contents/entries` with `Accept: application/vnd.github+json` header.
3. If 404: return `MemoryStats(gameCount: 0, ..., isAvailable: true)` — repo exists but entries/ dir is empty.
4. Decode response as `[GitHubDirectoryEntry]` (struct with `name: String, type: String`). Filter to `type == "file" && name.hasSuffix(".json")`.
5. For each game file, fetch raw content using `Accept: application/vnd.github.v3.raw`, decode as `[CollectiveMemoryEntry]`.
6. Aggregate: gameCount = number of game files, totalConfirmations = sum of all entry confirmations across all games.
7. Build recentContributions: collect all (slug, entry) pairs, sort by `lastConfirmed` descending (ISO 8601 lexicographic sort works), take top 10.
8. On any error during aggregation, return what was computed so far with `isAvailable: true` (partial data is better than no data).
9. Add comment noting this is not optimized for large repos (fetches each file sequentially).

**MemoryStatsService.fetchGameDetail(slug: String) -> GameDetail?:**
1. Get token. If `.unavailable`, return nil.
2. GET `https://api.github.com/repos/\(GitHubAuthService.shared.memoryRepo)/contents/entries/\(slug).json` with raw accept header.
3. Decode as `[CollectiveMemoryEntry]`.
4. Map each entry to `MemoryEntryViewData`, flattening the nested EnvironmentFingerprint fields and defaulting nil optionals to "".
5. On any failure (404, parse error, network), return nil.

**Use the same `performFetch` pattern as CollectiveMemoryService** — private static func with ResultBox + DispatchSemaphore + URLSession.shared.dataTask. Copy the pattern, do not try to share the private function.
