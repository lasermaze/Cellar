  <action>
Create a new file `Sources/cellar/Core/WikiIngestService.swift` with a `struct WikiIngestService` containing:

1. **`static func ingest(gameName: String) async -> Bool`** — the main pipeline:
   a. Fetch CompatibilityReport via `CompatibilityService.fetchReport(for: gameName)` (Lutris + ProtonDB).
   b. Fetch WineHQ fixes: construct search URL `https://appdb.winehq.org/objectManager.php?sClass=application&sTitle={urlEncoded}`, fetch HTML, use `WineHQParser().parseHTML(html, url: url)` to extract fixes. On any failure (network, parse, empty), treat as nil.
   c. Fetch PCGamingWiki fixes: construct URL `https://www.pcgamingwiki.com/wiki/{Game_Name_With_Underscores}` (replace spaces with underscores, percent-encode for URL path). Fetch HTML, use `PCGamingWikiParser().parseHTML(html, url: url)` to extract fixes. On any failure, treat as nil.
   d. If all sources returned nil/empty, print a skip message to stderr and return false.
   e. Format the page content using `formatGamePage(...)` (see below).
   f. Generate slug via `WikiService.slugify(gameName)`.
   g. POST the full page content to `games/{slug}.md` via `WikiService.postWikiAppend(page:entry:commitMessage:)`.
   h. Return true.

2. **`private static func formatGamePage(...) -> String`** — formats the markdown page:
   - Parameters: gameName, report (CompatibilityReport?), wineHQFixes (ExtractedFixes?), pcgwFixes (ExtractedFixes?)
   - Output format (plain markdown):
     ```
     # {Game Name}

     **Last updated:** {yyyy-MM-dd} | Sources: {comma-separated list of sources that had data}

     ## Compatibility

     **ProtonDB:** {tier} ({confidence}, {total} reports)
     (or "No ProtonDB data available" if nil)

     ## Known Working Configuration (Lutris)

     **Environment variables:**
     {list each as `- KEY=VALUE` or "(none)"}

     **DLL overrides:**
     {list each as `- name: native,builtin` or "(none)"}

     **Winetricks:**
     {list each as `- verb` or "(none)"}

     ## Fixes (WineHQ / PCGamingWiki)

     **Environment variables:**
     {combined list from both sources, deduplicated by key, or "(none)"}

     **DLL overrides:**
     {combined list, deduplicated by name, or "(none)"}

     **Winetricks:**
     {combined list, deduplicated, or "(none)"}

     **INI changes:**
     {from PCGW only, or "(none)"}
     ```

3. **`private static func fetchHTML(from url: URL) async -> String?`** — helper that does a URLSession GET, returns HTML string body on 200, nil otherwise. Timeout: 15 seconds.

For WineHQ URL construction, use `gameName.addingPercentEncoding(withAllowedCharacters: .urlQueryAllowed)` since it's a query parameter.
For PCGamingWiki URL construction, replace spaces with underscores then use `.addingPercentEncoding(withAllowedCharacters: .urlPathAllowed)`.

Import Foundation and SwiftSoup (for PageParser usage — SwiftSoup is already an SPM dependency).
