Create `Sources/cellar/Commands/WikiCommand.swift` with:

1. **`struct WikiCommand: ParsableCommand`** — parent command group:
   - `commandName: "wiki"`
   - `abstract: "Wiki management commands"`
   - `subcommands: [IngestCommand.self]`
   - No `run()` method (parent group only).

2. **`struct IngestCommand: AsyncParsableCommand`** — the ingest subcommand:
   - `commandName: "ingest"`
   - `abstract: "Pre-compile game wiki pages from external sources"`
   - Arguments/flags:
     - `@Argument(help: "Game name to ingest") var gameName: String?`
     - `@Flag(name: .long, help: "Ingest top games from Lutris catalog") var popular: Bool = false`
     - `@Flag(name: .long, help: "Ingest all games in local success database") var allLocal: Bool = false`
   - `mutating func validate() throws`:
     - Count how many modes are active (gameName != nil, popular, allLocal).
     - If 0: throw `ValidationError("Provide a game name, --popular, or --all-local")`
     - If >1: throw `ValidationError("Provide only one of: game name, --popular, --all-local")`
   - `mutating func run() async throws`:
     - **Single game mode:** Call `WikiIngestService.ingest(gameName: name)`, print success/skip message.
     - **Popular mode:** Call `CompatibilityService.fetchPopularGames(limit: 50)`. If empty, print error and return. Loop over games, call `WikiIngestService.ingest(gameName:)` for each, print progress (`[N/total] Game Name...`). Sleep 2 seconds between games via `try? await Task.sleep(nanoseconds: 2_000_000_000)`.
     - **All-local mode:** Call `SuccessDatabase.loadAll()`. If empty, print message and return. Loop over records, call `WikiIngestService.ingest(gameName: record.gameName)` for each, print progress. Sleep 2 seconds between games.
     - At the end, print summary: "Ingested N games (M skipped)".

Import ArgumentParser and Foundation.
