Create `Sources/cellar/Core/CollectiveMemoryService.swift` with a `struct CollectiveMemoryService` containing:

**Public API — single entry point:**

```swift
static func fetchBestEntry(
    for gameName: String,
    wineURL: URL
) -> String?  // Returns formatted context block string, or nil if no compatible entry / unavailable
```

**Implementation steps inside `fetchBestEntry`:**

1. **Auth check:** Call `GitHubAuthService.shared.getToken()`. If result is `.unavailable`, return nil silently (no logging needed for auth skip — this is expected when credentials aren't configured).

2. **Detect local Wine version:** Run `wine --version` using `Process()` (same pattern as WineProcess.run). Parse output format `"wine-9.0 (Staging)"` or `"wine-10.3"`:
   - Split on `"-"`, drop first component (`"wine"`)
   - Take remaining joined, split on `" "`, take first → `"9.0"` or `"10.3"`
   - If process fails or parsing returns nil, return nil (skip memory lookup).

3. **Detect Wine flavor:** Check if GPTK binary exists at `/usr/local/bin/gameportingtoolkit` or `/opt/homebrew/bin/gameportingtoolkit` using `FileManager.default.fileExists`. Return `"game-porting-toolkit"` if found, otherwise `"wine-stable"`.

4. **Build GitHub Contents API request:**
   - URL: `https://api.github.com/repos/{GitHubAuthService.shared.memoryRepo}/contents/entries/{slugify(gameName)}.json`
   - Headers: `Authorization: Bearer {token}`, `Accept: application/vnd.github.v3.raw`, `X-GitHub-Api-Version: 2022-11-28`
   - Set `request.timeoutIntervalForRequest = 5` (5-second timeout per locked decision)

5. **Synchronous fetch** using the DispatchSemaphore pattern from GitHubAuthService.performHTTPRequest:
   - Create a `ResultBox` final class with `@unchecked Sendable` to hold result
   - `URLSession.shared.dataTask(with: request)` completion handler fills box, signals semaphore
   - `semaphore.wait()`
   - **CRITICAL:** Check HTTP status code. If 404, return nil (no entry yet — normal). If >= 400 or network error, return nil silently. Only proceed on 200.

6. **Decode:** `try JSONDecoder().decode([CollectiveMemoryEntry].self, from: data)`. If decode fails or array is empty, return nil.

7. **Filter by arch (hard incompatible per locked decision):**
   ```swift
   #if arch(arm64)
   let localArch = "arm64"
   #else
   let localArch = "x86_64"
   #endif
   let archFiltered = entries.filter { $0.environment.arch == localArch }
   ```
   If empty after filtering, return nil.

8. **Rank — highest confirmations, tiebreak by Wine version proximity:**
   ```swift
   let localMajor = majorVersion(from: localWineVersion) ?? 0
   let ranked = archFiltered.sorted { a, b in
       if a.confirmations != b.confirmations { return a.confirmations > b.confirmations }
       let aMaj = majorVersion(from: a.environment.wineVersion) ?? 0
       let bMaj = majorVersion(from: b.environment.wineVersion) ?? 0
       return abs(aMaj - localMajor) < abs(bMaj - localMajor)
   }
   let best = ranked[0]
   ```

9. **Assess staleness and flavor:**
   - `isStale = (localMajor - entryMajor) > 1` where `entryMajor = majorVersion(from: best.environment.wineVersion)`
   - `flavorMismatch = best.environment.wineFlavor != localFlavor`

10. **Format context block** using `formatMemoryContext` (see below) and return the string.

**Private helpers:**

- `detectWineVersion(wineURL: URL) -> String?` — Run `Process()` with `wineURL` and `["--version"]`, pipe stdout, parse as described in step 2. Wrap everything in do/catch, return nil on any failure.

- `detectWineFlavor(wineURL: URL) -> String` — Check FileManager for GPTK binary at the two known paths. Return `"game-porting-toolkit"` or `"wine-stable"`.

- `majorVersion(from versionString: String) -> Int?` — Split on `" "` take first, split on `"."` take first, convert to Int.

- `performFetch(request: URLRequest) -> (data: Data, statusCode: Int)?` — DispatchSemaphore sync HTTP pattern. Return nil on network error. Return (data, statusCode) on any HTTP response.

- `formatMemoryContext(_ entry: CollectiveMemoryEntry, isStale: Bool, flavorMismatch: Bool, localWineVersion: String) -> String` — Build the context block:

```
--- COLLECTIVE MEMORY ---
A community-verified configuration exists for this game. Try it first before researching from scratch.

Confirmations: {N} | Verified environment: {arch}, Wine {wineVersion}, macOS {macosVersion}
[FLAVOR WARNING: Entry was verified with {entryFlavor}; local Wine flavor is {localFlavor}. Config may still apply.]
[STALENESS WARNING: Entry confirmed on Wine {entryMajor}.x; current Wine is {localMajor}.x ({diff} major versions ahead). Verify compatibility.]

Working Config:
  Environment variables:
    {KEY}={VALUE}  (one per line, or "(none)" if empty)
  DLL overrides:
    {dll} -> {mode} ({source})  (one per line, or "(none)" if empty)
  Registry:
    {key} {valueName} = {data}  (one per line, or "(none)" if empty)
  Launch args: {args joined by space, or "(none)"}
  Setup deps: {deps joined by comma, or "(none)"}

Agent's Reasoning (from prior session):
  "{reasoning}"
--- END COLLECTIVE MEMORY ---
```

Flavor warning line only included if `flavorMismatch` is true. Staleness warning line only included if `isStale` is true.

**Important implementation notes:**
- All errors are swallowed — the function returns `String?` and nil means "proceed without memory context"
- No local caching per locked decision
- No logging of errors to user — internal `print()` for debug is acceptable but not required
- The entire function should be wrapped in a top-level guard/nil-coalescing pattern so any unexpected failure returns nil
