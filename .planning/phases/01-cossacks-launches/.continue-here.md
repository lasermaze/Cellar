<decisions_made>

- Phase 1 needs agentic capabilities NOW, not deferred to Phase 2/3. The "hardcoded recipe + linear pipeline" approach doesn't work in practice.
- WineProcess.run() should return a WineResult struct (exitCode, captured stderr, elapsed time) — enables error analysis without changing streaming behavior.
- BottleScanner should skip Wine system dirs (windows/, programdata/, users/) and known non-game exes (unins000.exe, vcredist, etc.)
- WineErrorParser uses pattern matching on stderr (regex for known Wine error patterns) — no AI needed for Phase 1.
- Recipe schema additions: setup_deps (winetricks verbs), install_dir (expected install path), retry_variants (alternative env configs to try on failure)
- GameEntry.installPath replaced with executablePath (discovered, not empty string)
- Cossacks recipe needs setup_deps: ["dotnet48"] for GOG Galaxy installer
- The --no-quarantine flag was removed from Homebrew entirely — GuidedInstaller must install without it
</decisions_made>
